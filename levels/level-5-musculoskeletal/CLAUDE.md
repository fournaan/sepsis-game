# CLAUDE.md — Level 5: Musculoskeletal System

## Setting
Inside bones, joints, and soft tissue. Sub-levels span long bones, spine, joints, and ear canal.
Terrain varies: dense cortical bone → spongy cancellous bone → synovial fluid (swim level).

## Unique Mechanic
Terrain fragility — osteoporotic ground can give way; muscles act as bounce pads.

## Enemies
- **Henchmen:** Staph aureus (same species, new setting)
- **Boss:** Staph aureus / MRSA (osteomyelitis / PJI context)

## Obstacles
| Obstacle | Effect |
|----------|--------|
| Fractures | Gaps to jump across |
| Scoliosis | Tilted, uneven terrain |
| Osteoporosis | Fragile ground that crumbles underfoot |
| Ligament tears | Floor sections that break away on contact |
| Synovial fluid | Swim level inside an infected joint |
| Muscle terrain | Bouncy surface — affects jump height |

## Antibiotic Unlocks
- Flucloxacillin (bone Staph A — early reward)
- Vancomycin (MRSA bone/joint — boss reward)
- Vitamin D unlockable as a bone-health tool (non-antibiotic)
